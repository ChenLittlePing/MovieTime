# MovieTime/电影时光
A app to look over recent movies information, that just for studying iOS development.

(热播电影查看App，用于学习iOS开发)

![演示](https://github.com/ChenLittlePing/MovieTime/blob/master/Demo/demo.gif)

# 功能：
* 查看当前热映电影
* 查看电影详情
   * 电影简介
   * 电影海报
   * 导演/演员
   * 预告片
   * 影评

# 使用框架：

* AFNetworking
* SDWebImage
* Mantle
